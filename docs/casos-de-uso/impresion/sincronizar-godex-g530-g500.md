# Solución de problemas con impresoras de etiquetas Godex G500/G530

## Sincronización de rollo de etiquetas

Cuando la impresora es nueva o se cambia el tamaño/modelo de las etiquetas, la impresora ha de ser recalibrada para que reconozca correctamente las etiquetas.

Para realizar la sincronización sigue estos pasos:

1. Apagar impresora
2. Presionar el botón feed y mantener presionado mientras se enciende la impresora
3. Cuando el botón Feed cambia a color "naranja" soltarlo.
4. La impresora realizará unos movimientos y terminará imprimiendo una o varias etiquetas.
5. La calibración ha terminado.
