# Activar avisos audiovisuales

El servicio de avisos de matrícula, está disponible si está contratado el módulo Avisos Audiovisuales.

Para poner en marcha el servicio de avisos mediante panel LED y megafonía, procede de la siguiente forma:

Iniciar la aplicación CREATIVA3D ITV

Ir al menú Gestión / Avisos Audiovisuales / Controlador paneles de aviso
